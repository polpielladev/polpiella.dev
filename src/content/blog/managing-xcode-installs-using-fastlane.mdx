A new action called `xcodes` is now available in Fastlane versions bigger than 2.211.0. This actions is a wrapper around the `xcodes` command line tool, which allows you to manage multiple Xcode versions on your machine.

<StaticTweet id="1591561723794030592" />

## Installing `xcodes`

The `xcodes` command line tool is available on Homebrew, and can be installed it by running:

```bash
brew install robotsandpencils/made/xcodes
```
